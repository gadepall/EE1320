\begin{problem}
Connect the RaspberryPi to the desktop and power it on. 
%Make sure that box is open and able to use GPIO pins
\end{problem}

\begin{problem}
Open 
\end{problem}
\begin{lstlisting}
sudo nano /boot/cmdline.txt
\end{lstlisting}
and remove  \textbf{console=serial0,115200} or \textbf{console=ttyAMA0,115200}.  Don't delete anything else in the file. \textbf{Ctrl+X} and exit.
\begin{problem}
In order to enable GPIO serial port, 
\begin{lstlisting}
sudo nano /boot/config.txt
\end{lstlisting}
and add 
\begin{lstlisting}
enable_uart=1
\end{lstlisting}
at the end of the file.  Reboot.
\end{problem}
Use \textbf{/dev/ttyS0} in any code which accesses the Serial Port of RaspberryPi.
